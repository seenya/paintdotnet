# paintdotnet-cookbook

This cookbook installs Paint.Net on the windows platform.

## Supported Platforms

Windows

## Usage

### paintdotnet::default

Include `paintdotnet` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[paintdotnet::default]"
  ]
}
```
